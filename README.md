## 📖 About This Repository

This repository is the **official class codebase** for Group A's React course. ## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed before running any project:

- **Node.js** v18 or higher — [Download](https://nodejs.org/)
- **npm** v9+ (comes with Node) or **yarn**
- A code editor (recommended: [VS Code](https://code.visualstudio.com/))

### Run Any Class Project Locally

```bash
# 1. Clone the repository
git clone https://github.com/mrinal1224/Group-A-2029-React-Course.git

# 2. Navigate into the class folder of your choice
cd "Group-A-2029-React-Course/Class-5 (imdb-Clone)"

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Then open your browser at `http://localhost:5173` 🎉
